Run (in Debian 10):

```
wget https://raw.githubusercontent.com/rezamarzban/reinstall/refs/heads/main/other/setup.sh
chmod +x setup.sh
./setup.sh

```

Each time, To reactivate Live Alpine (Rescue System) at next boot, run:

```
./reactivate.sh

```
