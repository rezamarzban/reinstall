Remote Alpine from Debian 


Run in Debian 10:

```
wget https://raw.githubusercontent.com/rezamarzban/reinstall/refs/heads/main/other/setup.s
chmod +x setup.sh
./setup.sh

```

To reactivate Live Alpine at next boot, run:

```
./reactivate.sh

```

Username: `root`
